# 运动控制与视觉系统预留设计

> 本文档定义运动控制和视觉系统的预留接口和架构设计，为未来扩展预留空间。

---

## 一、设计目标

### 1.1 运动控制目标

- **多品牌支持**：支持 Beckhoff TwinCAT、Omron NJ/NX、Siemens S120 等主流运动控制器
- **单轴控制**：支持点位运动、速度控制、回零等单轴操作
- **多轴联动**：支持电子齿轮、电子凸轮、插补运动等多轴协调
- **轨迹规划**：支持直线、圆弧、样条等轨迹插补
- **安全监控**：支持软硬限位、跟随误差、运动状态监控

### 1.2 视觉系统目标

- **多相机支持**：支持 GigE Vision、USB3 Vision、模拟相机等多种相机类型
- **算法工具**：支持边缘检测、模板匹配、Blob分析、条码识别等视觉算法
- **检测流程**：支持检测任务编排、多工具串联、结果判定
- **与运动集成**：支持视觉引导运动、飞拍、同步触发等应用

### 1.3 集成目标

- **与脚本引擎集成**：支持在脚本中调用运动控制和视觉功能
- **与 HMI 集成**：支持在组态界面显示运动状态和视觉结果
- **与报警集成**：支持运动异常和视觉检测结果的报警处理

```csharp
/// <summary>
```

### 2.3 运动控制器基类

```csharp
namespace Acme.Industrial.Communication.Motion.Core;

/// <summary>
/// 运动控制器基类。
/// </summary>
public abstract class MotionControllerBase : IMotionController
{
    protected readonly IAppLogger _logger;
    protected readonly List<IMotionAxis> _axes = new();
    protected readonly Dictionary<string, IMotionAxisGroup> _axisGroups = new();

    public string Id { get; protected set; } = "";
    public string Name { get; protected set; } = "";
    public abstract string ControllerType { get; }
    public ConnectionState ConnectionState { get; protected set; } = ConnectionState.Disconnected;
    public int AxisCount => _axes.Count;
    public IReadOnlyList<IMotionAxis> Axes => _axes.AsReadOnly();

    public event EventHandler<ConnectionStateChangedEventArgs>? ConnectionStateChanged;
    public event EventHandler<AxisStateChangedEventArgs>? AxisStateChanged;

    protected MotionControllerBase(IAppLoggerFactory loggerFactory)
    {
        _logger = loggerFactory.CreateLogger<MotionControllerBase>();
    }

    public abstract Task<OperateResult> ConnectAsync(CancellationToken ct = default);
    public abstract Task<OperateResult> DisconnectAsync(CancellationToken ct = default);
    public abstract Task<OperateResult> EmergencyStopAsync(CancellationToken ct = default);
    public abstract Task<OperateResult> EnableAllAsync(CancellationToken ct = default);
    public abstract Task<OperateResult> DisableAllAsync(CancellationToken ct = default);
    
    public abstract Task<OperateResult<T>> ReadControllerVariableAsync<T>(string variableName,
                                                                         CancellationToken ct = default);
    public abstract Task<OperateResult> WriteControllerVariableAsync<T>(string variableName, T value,
                                                                          CancellationToken ct = default);
    public abstract Task<OperateResult> ExecuteProgramAsync(string programName,
                                                              CancellationToken ct = default);

    public IMotionAxis? GetAxis(int axisNumber) =>
        _axes.FirstOrDefault(a => a.AxisNumber == axisNumber);

    public IMotionAxisGroup? GetAxisGroup(string groupName) =>
        _axisGroups.TryGetValue(groupName, out var group) ? group : null;

    protected void RaiseConnectionStateChanged(ConnectionState oldState, ConnectionState newState)
    {
        ConnectionStateChanged?.Invoke(this, new ConnectionStateChangedEventArgs
        {
            ControllerId = Id,
            OldState = oldState,
            NewState = newState
        });
    }

    protected void RaiseAxisStateChanged(int axisNumber, AxisState oldState, AxisState newState)
    {
        AxisStateChanged?.Invoke(this, new AxisStateChangedEventArgs
        {
            AxisNumber = axisNumber,
            OldState = oldState,
            NewState = newState
        });
    }

    public virtual void Dispose()
    {
        foreach (var axis in _axes)
            axis.Dispose();
        foreach (var group in _axisGroups.Values)
            group.Dispose();
    }
}
```

### 2.4 Beckhoff TwinCAT 实现示例

```csharp
namespace Acme.Industrial.Communication.Motion.Beckhoff;

/// <summary>
/// Beckhoff TwinCAT ADS 运动控制器实现。
/// </summary>
public class BeckhoffMotionController : MotionControllerBase
{
    private IAdsClient? _adsClient;
    private readonly string _ipAddress;
    private readonly int _amsNetId;
    
    // TwinCAT ADS 地址
    private const ushort Port = 851;  // NC 轴任务端口
    
    public override string ControllerType => "Beckhoff TwinCAT";
    
    public BeckhoffMotionController(string id, string ipAddress, IAppLoggerFactory loggerFactory)
        : base(loggerFactory)
    {
        Id = id;
        Name = $"TwinCAT@{ipAddress}";
        _ipAddress = ipAddress;
        _amsNetId = AdsClient.ParseAmsNetId(ipAddress);
    }
    
    public override async Task<OperateResult> ConnectAsync(CancellationToken ct = default)
    {
        try
        {
            var oldState = ConnectionState;
            ConnectionState = ConnectionState.Connecting;
            RaiseConnectionStateChanged(oldState, ConnectionState);
            
            _adsClient = new AdsClient();
            await _adsClient.ConnectAsync(_amsNetId, Port);
            
            // 读取轴数量并初始化轴
            await InitializeAxesAsync();
            
            ConnectionState = ConnectionState.Connected;
            RaiseConnectionStateChanged(ConnectionState.Connecting, ConnectionState);
            _logger.Info($"TwinCAT 连接成功: {_ipAddress}");
            
            return OperateResult.Ok();
        }
        catch (Exception ex)
        {
            _logger.Error($"TwinCAT 连接失败: {ex.Message}", ex);
            ConnectionState = ConnectionState.Error;
            return OperateResult.Fail(ErrorCode.CommConnectFailed, ex.Message);
        }
    }
    
    private async Task InitializeAxesAsync()
    {
        // 读取轴数量（ TwinCAT 变量：nAxisCount）
        var result = await ReadControllerVariableAsync<int>("nAxisCount");
        if (!result.IsSuccess) return;
        
        var axisCount = result.Content;
        for (int i = 1; i <= axisCount; i++)
        {
            var axis = new BeckhoffMotionAxis(_adsClient!, i, _logger);
            _axes.Add(axis);
        }
    }
    
    public override async Task<OperateResult> EmergencyStopAsync(CancellationToken ct = default)
    {
        _logger.Warn("执行紧急停止");
        
        // 写入急停变量
        await _adsClient!.WriteAsync("MAIN.bEStop", true);
        
        return OperateResult.Ok();
    }
    
    // ... 其他方法实现
}

/// <summary>
/// Beckhoff 单轴实现。
/// </summary>
public class BeckhoffMotionAxis : MotionAxisBase
{
    private readonly IAdsClient _adsClient;
    
    public BeckhoffMotionAxis(IAdsClient adsClient, int axisNumber, IAppLoggerFactory loggerFactory)
        : base(axisNumber, loggerFactory)
    {
        _adsClient = adsClient;
    }
    
    protected override async Task<double> ReadPositionAsync(CancellationToken ct)
    {
        var result = await _adsClient.ReadAsync<double>($"Axes.Axis{AxisNumber}.actPos");
        return result.Content;
    }
    
    protected override async Task MoveAbsoluteInternalAsync(double position, double velocity,
                                                            CancellationToken ct)
    {
        // 写入目标位置和速度，启动运动
        await _adsClient.WriteAsync($"Axes.Axis{axisNumber}.setPos", position);
        await _adsClient.WriteAsync($"Axes.Axis{axisNumber}.setVel", velocity);
        await _adsClient.WriteAsync($"Axes.Axis{axisNumber}.bMoveAbs", true);
    }
    
    // ... 其他方法实现
}
```

```csharp
        {
```

```csharp
            await _client.SetFloatValueAsync("GainAuto", 0);     // 手动增益
            
            ConnectionState = CameraConnectionState.Connected;
            RaiseStateChanged(CameraConnectionState.Connecting, ConnectionState);
            
            _logger.Info($"GigE 相机连接成功: {Name}");
            return OperateResult.Ok();
        }
        catch (Exception ex)
        {
            _logger.Error($"GigE 相机连接失败: {ex.Message}", ex);
            ConnectionState = CameraConnectionState.Error;
            return OperateResult.Fail(ErrorCode.CommConnectFailed, ex.Message);
        }
    }
    
    public async Task<OperateResult<ImageData>> GrabAsync(CancellationToken ct = default)
    {
        if (ConnectionState != CameraConnectionState.Connected)
            return OperateResult.Fail<ImageData>("相机未连接");
        
        try
        {
            var rawImage = await _client!.GrabImageAsync(ct);
            
            var imageData = new ImageData

                Height = rawImage.Height,
                PixelFormat = rawImage.PixelFormat,
                Bytes = rawImage.Data,
                Timestamp = DateTime.Now
            };

        }
    }

            OldState = oldState,
            NewState = newState
        });
```

```csharp
namespace Acme.Industrial.Services.MotionControl;

/// <summary>
/// 运动协调器 - 协调多轴运动，支持运动脚本。
/// </summary>
public class MotionCoordinator
{
    private readonly IMotionController _controller;
    private readonly IAppLogger _logger;
    private readonly Dictionary<string, MotionTask> _motionTasks = new();
    
    public MotionCoordinator(IMotionController controller, IAppLoggerFactory loggerFactory)
    {
        _controller = controller;
        _logger = loggerFactory.CreateLogger<MotionCoordinator>();
    }
    
    /// <summary>
    /// 执行点位运动。
    /// </summary>
    public async Task<OperateResult> MoveToAsync(string axisName, double position,
                                                  double velocity, CancellationToken ct = default)
    {
        var axis = _controller.Axes.FirstOrDefault(a => a.Name == axisName);
        if (axis == null)
            return OperateResult.Fail($"轴不存在: {axisName}");
        
        _logger.Info($"轴 {axisName} 移动到 {position}");
        return await axis.MoveAbsoluteAsync(position, velocity, ct);
    }
    
    /// <summary>
    /// 等待轴运动完成。
    /// </summary>
    public async Task<OperateResult> WaitForIdleAsync(string axisName, 
                                                       TimeSpan timeout,
                                                       CancellationToken ct = default)
    {
        var axis = _controller.Axes.FirstOrDefault(a => a.Name == axisName);
        if (axis == null)
            return OperateResult.Fail($"轴不存在: {axisName}");
        
        var deadline = DateTime.Now + timeout;
        while (DateTime.Now < deadline)
        {
            if (ct.IsCancellationRequested)
                return OperateResult.Fail(ErrorCode.Cancelled, "操作已取消");
            
            if (axis.State == AxisState.Ready)
                return OperateResult.Ok();
            
            await Task.Delay(10, ct);
        }
        
        return OperateResult.Fail(ErrorCode.Timeout, $"轴 {axisName} 等待超时");
    }
    
    /// <summary>
    /// 执行轨迹运动（多点路径）。
    /// </summary>
    public async Task<OperateResult> ExecuteTrajectoryAsync(string groupName,
                                                            IReadOnlyList<TrajectoryPoint> points,
                                                            CancellationToken ct = default)
    {
        var group = _controller.GetAxisGroup(groupName);
        if (group == null)
            return OperateResult.Fail($"轴组不存在: {groupName}");
        
        _logger.Info($"执行轨迹运动，共 {points.Count} 个点");
        
        foreach (var point in points)
        {
            var result = await group.LinearInterpolationAsync(point.Positions, point.Velocity, ct);
            if (!result.IsSuccess)
                return result;
            
            // 等待到达点位
            await WaitGroupIdleAsync(groupName, TimeSpan.FromSeconds(10), ct);
        }
        
        return OperateResult.Ok();
    }
}
```

```csharp
namespace Acme.Industrial.Services.MotionControl;

/// <summary>
/// 运动安全监控器。
/// </summary>
public class SafetyMonitor
{
    private readonly IMotionController _controller;
    private readonly IEventBus _eventBus;
    private readonly IAppLogger _logger;
    private readonly Timer _monitorTimer;
    private readonly SafetyLimits _limits;
    
    public SafetyMonitor(
        IMotionController controller,
        IEventBus eventBus,
        SafetyLimits limits,
        IAppLoggerFactory loggerFactory)
    {
        _controller = controller;
        _eventBus = eventBus;
        _limits = limits;
        _logger = loggerFactory.CreateLogger<SafetyMonitor>();
        
        // 每 50ms 检查一次
        _monitorTimer = new Timer(CheckSafety, null, 50, 50);
        
        // 订阅轴状态变更
        _controller.AxisStateChanged += OnAxisStateChanged;
    }
    
    private void CheckSafety(object? state)
    {
        foreach (var axis in _controller.Axes)
        {
            // 检查跟随误差
            if (Math.Abs(axis.FollowingError) > _limits.FollowingErrorLimit)
            {
                _logger.Error($"轴 {axis.AxisNumber} 跟随误差超限: {axis.FollowingError}");
                _ = _controller.EmergencyStopAsync();
                _eventBus.PublishAsync(new MotionSafetyAlarmEvent
                {
                    AxisNumber = axis.AxisNumber,
                    AlarmType = SafetyAlarmType.FollowingError,
                    Message = $"跟随误差超限: {axis.FollowingError}"
                });
                return;
            }
            
            // 检查限位
            if (axis.PositiveLimit || axis.NegativeLimit)
            {
                _logger.Warn($"轴 {axis.AxisNumber} 触发限位");
                _eventBus.PublishAsync(new MotionSafetyAlarmEvent
                {
                    AxisNumber = axis.AxisNumber,
                    AlarmType = axis.PositiveLimit 
                        ? SafetyAlarmType.PositiveLimit 
                        : SafetyAlarmType.NegativeLimit,
                    Message = "触发限位"
                });
            }
        }
    }
    
    private void OnAxisStateChanged(object? sender, AxisStateChangedEventArgs e)
    {
        if (e.NewState == AxisState.Error)
        {
            _logger.Error($"轴 {e.AxisNumber} 进入错误状态");
            _eventBus.PublishAsync(new MotionSafetyAlarmEvent
            {
                AxisNumber = e.AxisNumber,
                AlarmType = SafetyAlarmType.AxisError,
                Message = "轴进入错误状态"
            });
        }
    }
    
    public void Dispose()
    {
        _monitorTimer.Dispose();
        _controller.AxisStateChanged -= OnAxisStateChanged;
    }
}
```

```csharp
namespace Acme.Industrial.Services.Vision;

/// <summary>
/// 检测服务 - 执行视觉检测任务。
/// </summary>
public class InspectionService
{
    private readonly IVisionSystem _visionSystem;
    private readonly IEventBus _eventBus;
    private readonly IAppLogger _logger;
    private readonly Dictionary<string, VisionTask> _tasks = new();
    
    public InspectionService(
        IVisionSystem visionSystem,
        IEventBus eventBus,
        IAppLoggerFactory loggerFactory)
    {
        _visionSystem = visionSystem;
        _eventBus = eventBus;
        _logger = loggerFactory.CreateLogger<InspectionService>();
    }
    
    /// <summary>
    /// 注册检测任务。
    /// </summary>
    public void RegisterTask(VisionTask task)
    {
        _tasks[task.Id] = task;
        _logger.Info($"检测任务已注册: {task.Id}");
    }
    
    /// <summary>
    /// 执行检测。
    /// </summary>
    public async Task<InspectionResult> InspectAsync(string taskId, 
                                                     string cameraId,
                                                     CancellationToken ct = default)
    {
        if (!_tasks.TryGetValue(taskId, out var task))
            return new InspectionResult { IsSuccess = false, Message = "任务不存在" };
        
        var camera = _visionSystem.GetCamera(cameraId);
        if (camera == null)
            return new InspectionResult { IsSuccess = false, Message = "相机不存在" };
        
        // 1. 采集图像
        var grabResult = await camera.GrabAsync(ct);
        if (!grabResult.IsSuccess)
            return new InspectionResult { IsSuccess = false, Message = grabResult.Message };
        
        var image = grabResult.Content;
        
        // 2. 执行检测
        var result = new InspectionResult
        {
            TaskId = taskId,
            CameraId = cameraId,
            ImageTime = image.Timestamp,
            IsSuccess = true
        };
        
        foreach (var step in task.Steps)
        {
            var tool = _visionSystem.Tools.FirstOrDefault(t => t.Name == step.ToolName);
            if (tool == null)
            {
                result.IsSuccess = false;
                result.Message = $"工具不存在: {step.ToolName}";
                break;
            }
            
            var roi = step.RegionOfInterest;
            var stepResult = tool.Execute(image, roi);
            
            result.StepResults.Add(new StepResult
            {
                ToolName = step.ToolName,
                Result = stepResult,
                ExecutionTimeMs = stepResult.ExecutionTimeMs
            });
            
            if (!stepResult.IsSuccess)
            {
                result.IsSuccess = false;
                result.Message = stepResult.ErrorMessage;
            }
        }
        
        // 3. 发布结果事件
        await _eventBus.PublishAsync(new InspectionCompletedEvent
        {
            TaskId = taskId,
            Result = result
        });
        
        return result;
    }
}
```

## 七、集成点

### 7.1 与脚本引擎集成

运动控制和视觉功能可以通过脚本引擎访问：

```csharp
// 脚本中访问运动控制
var axis = Motion.GetAxis("X轴");
await axis.MoveAbsoluteAsync(100.0, 50.0);

// 脚本中访问视觉
var camera = Vision.GetCamera("主相机");
var image = await camera.GrabAsync();
var result = Vision.FindEdge(image, region);
```

### 7.2 与 HMI 集成

- 运动状态：通过 TagBinding 显示轴位置、速度、状态
- 视觉结果：通过点位写入检测结果，触发 HMI 显示
- 报警信息：通过事件总线发布报警，在 HMI 显示

### 7.3 与报警系统集成

```csharp
// 运动异常报警
_eventBus.Subscribe<MotionSafetyAlarmEvent>(async evt =>
{
    await _alarmService.RaiseAlarmAsync(new Alarm
    {
        Level = AlarmLevel.High,
        Source = $"Motion.{evt.AxisNumber}",
        Message = evt.Message
    });
});

// 视觉检测失败报警
_eventBus.Subscribe<InspectionCompletedEvent>(async evt =>
{
    if (!evt.Result.IsSuccess)
    {
        await _alarmService.RaiseAlarmAsync(new Alarm
        {
            Level = AlarmLevel.Medium,
            Source = $"Vision.{evt.Result.TaskId}",
            Message = evt.Result.Message ?? "检测失败"
        });
    }
});
```

## 八、小结

本文档为框架预留了运动控制和视觉系统的扩展能力：

- **运动控制**：支持多品牌运动控制器、单轴/多轴运动、轨迹规划、安全监控
- **视觉系统**：支持多相机、多种视觉工具、检测流程、结果集成
- **完整集成**：与脚本引擎、HMI、报警系统无缝集成

详细实现时，可根据具体品牌（如 Siemens、Omron）或视觉库（如 Halcon、VisionPro）编写对应的驱动实现。
